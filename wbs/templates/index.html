About This Project

The Open Source House Project is a Django-based Work Breakdown Structure (WBS) management system designed for residential construction projects. It provides comprehensive tools for project planning, scheduling, tracking, and collaboration.

Key Features: Interactive Gantt chart scheduling, Kanban board for agile workflows, hierarchical WBS structure, task dependencies, resource allocation, progress tracking, and data import/export capabilities.
